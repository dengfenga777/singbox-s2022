# sing-box s2022 一键安装脚本

## 功能
- 安装最新版 sing-box
- 配置 Shadowsocks 2022 协议（2022-blake3-aes-128-gcm）
- 自动生成随机密码
- 开启流量统计
- 提供 `ss2022-traffic` 命令查询流量

## 一键安装命令
```bash
bash <(curl -fsSL https://raw.githubusercontent.com/dengfenga777/singbox-s2022/main/s2022_auto.sh)
```

## 查看流量
```bash
ss2022-traffic
```
